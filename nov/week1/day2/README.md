String Method
================
->concat
->trim(remove the extra space in starting and ending)
->at(used to get the indexing value)
->charAt(used to get the indexing value)
->replace(accept 2 values {1.existing char and 2. replacing char })
->toString(convert to string)
->slice(give the substring from original string)
    ->accept 2 value
    ->start index and last index
    ->give the last index-1
    ->if 1 value is added then it take it as first index and go to last index
    ->starting index < ending index>
->split(break the sentence and store in array format)
